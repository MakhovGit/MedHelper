# MedHelper
MedHelper - приложение, которое хранит и отображает давление и пульс пользователя по времени. При этом данные хранятся 
в Google Firestore и не хранятся на самом устройстве.

Используемые технологии:
- MVVM
- Coroutines
- Koin
- Google Firestore
